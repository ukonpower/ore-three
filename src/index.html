Ore Three

three.js utility library.

Ore Threeとは？

Ore Threeはthree.jsをより手軽に、より有効に使用するための便利機能をまとめたライブラリです。
クリックやリサイズなどのイベントを自動で登録し、アニメーションループをいち早くコーディングできるController、
GPGPUを手軽に行うためのGPUComputationContorllerなどざまざまなクラスが用意されています。

インストール

npmで簡単にインストールすることができます。

npm install ore-three

ビルドされたものはこちらです。

以下のようにしてインポートします。

import * as ORE from 'ore-three'
